    • MRI SENSE Reconstruction Mapping

  • Used Matlab image processing toolbox and SENSE MRI module for reconstruction
  • Designed a algorithm for coil sensitivity mapping based on coil shapes
  • The custom sensitivity map function provides a flexible way to implement Sinusoidal, Cosine and Saw-tooth functions based on coil geometry and placement
  • The custom sensitivity map can then be used in the SENSE reconstruction algorithm provide to reconstruction based on the true geometry of the coil
  • An accurate sensitivity map can provide reduction in the image dataset and can reduce computational times
  • Key technologies used: Matlab Image processing toolbox, SENSE, MR-Imaging modalities

    • Image Processing Web Service

  • Used Windows Communication Foundation (WCF) and OpenCV with EMGU C# wrapper
  • The architecture was based around a WCF service running on the host PC and a simple C# forms app/Web app to consume it
  • The WCF service was used to host EMGU based OpenCV filter algorithms (FFT, De-noise, Face-recognition. etc)
  • The web app was run on a IIS express server with a simple forms interface to browse images, send images, select filter and view received images
  • The images sent through the web app and processed images from the WCF service were stored on a SQL Lite server on the host PC
  • Key technologies used: C#, WCF, OpenCV, IIS Express, Windows forms
